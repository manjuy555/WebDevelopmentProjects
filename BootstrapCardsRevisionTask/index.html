Classes List of Bootstrap Card

Quick reference for all Bootstrap card classes and what makes each one unique.

`
ClassWhat it doesKey Difference
.cardRoot card container, adds border, bg-white, border-radius, shadowAlways required
.card-bodyAdds padding inside the cardMain content area
.card-titleBold heading with bottom marginPrimary heading
.card-subtitleSmaller grey headingSecondary heading, uses muted color
.card-textParagraph; last child removes bottom marginCleans up spacing
.card-linkAdds margin-left to separate adjacent linksOnly for inline links
.card-headerTop strip with light grey bg + bottom borderOutside card-body, gets its own bg
.card-footerBottom strip with light grey bg + top borderOutside card-body, aligns at bottom in card-group
.card-img-topRounds only top cornersImage before card-body
.card-img-bottomRounds only bottom cornersImage after card-body
.card-imgRounds all corners, full width backgroundUsed with overlay
.card-img-overlayAbsolutely positions content over card-imgOnly works with .card-img
.card-groupCards side-by-side, merged borders, equal heightNo gutters between cards
.card-header-tabsMerges tab nav into card-header borderFlush with header bottom border
.card-header-pillsPill-style nav inside card-headerFloats inside header, no border flush
